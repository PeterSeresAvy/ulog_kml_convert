# ulog_kml_convert

## Setup:

Clone the repository or download the zip folder.

    python -m venv venv

Windows:

    venv\scripts\activate

Linux:

    source venv/scripts/activate

Packages:   

    pip install -r requirements.txt
    
## Usage:

Go to the project's folder and run:

    python convert
    
Or run __main__.py from an IDE.
